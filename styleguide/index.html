This site has a small style guide (or more precisely, pattern library) due to a focus on reusable components. The patterns have been grouped by scope, making it easy to understand where to use each.

The .s-text scope

Represents an area of user-generated content, where the HTML comes from a CMS. It will set some cross-screen typography styles. It will also privde styles for some classless components for an efortless styling experience.

Global scope

Here patterns inherit the default properties from the root element. Besides its own styles, patterns here will only be afected by the resets and some basic generic styles like font family and size.
